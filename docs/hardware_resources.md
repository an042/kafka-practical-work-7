# Аппаратные ресурсы кластера

## Kafka-брокеры

| Параметр | Значение |
|----------|---------|
| Тип инстанса | `s3-c2-m8` |
| CPU | 2 vCPU |
| RAM | 8 GB |
| Тип диска | `network-ssd` |
| Размер диска | 10 GB |
| Количество брокеров | 3 (по одному на зону доступности) |
| Зоны | ru-central1-a, ru-central1-b, ru-central1-d |

## ZooKeeper-ноды

| Параметр | Значение |
|----------|---------|
| Тип инстанса | `s3-c2-m8` |
| Тип диска | `network-ssd` |
| Размер диска | 10 GB |
| Количество нод | 3 (автоматически при brokers_count > 1) |

## Сеть

| Параметр | Значение |
|----------|---------|
| Сеть | `default` (enp7h15721pj53b44vn0) |
| Публичный IP | Да (`assign_public_ip = true`) |
| Порт клиентов | 9091 (SASL_SSL) |
| Порт Schema Registry | 443 (HTTPS) |

## Параметры топика `events`

| Параметр | Значение | Описание |
|----------|---------|----------|
| Партиции | 3 | Параллельная обработка по зонам |
| Replication Factor | 3 | Каждая партиция на всех брокерах |
| `log.cleanup.policy` | `delete` | Удалять старые сообщения |
| `log.retention.ms` | 604800000 | Хранить 7 дней |
| `log.segment.bytes` | 134217728 | Сегмент 128 МБ |
